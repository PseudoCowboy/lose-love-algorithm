# [345. Reverse Vowels of a String](https://leetcode.com/problems/reverse-vowels-of-a-string/)

## 题目

Write a function that takes a string as input and reverse only the vowels of a string.



Example 1:

```c
Input: "hello"
Output: "holle"
```

Example 2:

```c
Input: "leetcode"
Output: "leotcede"
```
